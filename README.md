# AI Virtual Assistant Using Python

## Project Description

This project is an AI Virtual Assistant developed using Python. It can accept both voice and text commands to perform common tasks such as opening websites, searching Wikipedia, telling the current time and date, opening desktop applications, and playing music.

## Features

- Voice Command Support
- Text Command Support
- Text-to-Speech
- Speech Recognition
- Open Google
- Open YouTube
- Search Google
- Search Wikipedia
- Tell Current Time
- Tell Current Date
- Open Notepad
- Open Calculator
- Play Music

## Technologies Used

- Python
- SpeechRecognition
- pyttsx3
- Wikipedia API
- Webbrowser Module
- Subprocess Module

## How to Run

1. Install Python.
2. Install the required libraries:

```
pip install -r requirements.txt
```

3. Run the project:

```
python main.py
```
